# Tarah App

**[Live Demo](https://tarah-app.vercel.app/)**

## Features

- **Text Objects**: Create text with real-time effects such as neon, glitch, echo, and contour, with full support for Persian fonts.  
- **Smart Images**: Upload images, apply HSL filters, remove backgrounds with AI, crop, and use masking.  
- **Geometric Shapes**: Draw vector paths, circles, and rectangles with advanced color and gradient controls.  
- **Digital Codes**: Generate QR codes and barcodes with customizable colors, borders, and shadows.  
- **Layer System**: Manage layers with depth, grouping, and boolean operation combinations for complex compositions.  
- **Performance Management**: Optimized with debounced transforms, throttled zoom, memory disposal, and canvas optimization.

## Technologies

- Vue 3  
- TypeScript  
- Fabric.js  
- Pinia  
- Element Plus
